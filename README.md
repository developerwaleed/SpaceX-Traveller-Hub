## Run Locally

Clone the project

```bash
  git clone git@github.com:caasperr/SpaceX-Traveller-Hub.git
```

Go to the project directory

```bash
  cd SpaceX-Traveller-Hub
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npx eslint . --fix
  npm run start
```
